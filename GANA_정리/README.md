# GANA 제품 정리 — 작업용 패키지

> 박소정 직접 정리용. Claude가 카탈로그/단가표/메모에서 긁어 한 곳에 모아둠.

## 폴더 구성

```
GANA_정리/
├─ 제품마스터.xlsx              ← ★ 메인. 전 제품 표 (제품군·이름·상세정보·인증현황·기타)
├─ 카탈로그_페이지_이미지/       ← 카탈로그 PDF를 페이지 이미지로 렌더 (이름·스펙 보면서 매칭용)
│   ├─ 웹카탈로그2021-01..91.png   (메인 카탈로그, 91p)
│   ├─ 신제품2024-01..16.png
│   └─ DMP플러스2025-01..07.png
└─ 추출_제품컷/                  ← 카탈로그에서 추출된 개별 제품 사진 (가리지 않고 전부)
    ├─ _웹사이트_확정18종_네이밍완료/   (이미 이름 붙은 18장 — 기준 샘플)
    ├─ 웹카탈로그2021_컷/        (183장, 라벨 없음 img-000.jpg…)
    ├─ 신제품2024_컷/            (17장)
    ├─ DMP플러스2025_컷/         (9장)
    ├─ RND가격표2026_컷/         (4장)
    └─ 브랜드_일반/              (11장)
```

## 엑셀 보는 법 (제품마스터.xlsx)
- **시트 1 「제품마스터」**: 전 제품 55행. 상단 2행 고정 + 필터 걸려 있음 → 헤더의 ▼로 판매상태별/제품군별 정렬 가능.
- **시트 2 「범례·안내」**: 색상 의미 + 인증번호 출처 정리.
- **판매상태 색상**: 🟢판매확정(18) · 🟡확인필요(3) · 🟠판매여부결정(4) · 🔵ODM전용(7) · ⬜제외(23, 필러·클리닉케어·R&D)

## 사진 매칭 흐름 (추천)
1. `제품마스터.xlsx`에서 제품 한 줄을 본다 → "사진(출처/파일)" 칸에 어느 카탈로그인지 적혀 있음.
2. `카탈로그_페이지_이미지/`에서 그 카탈로그를 넘기며 해당 제품 페이지를 찾는다 (이름·스펙 같이 보임).
3. `추출_제품컷/`의 같은 카탈로그 폴더에서 그 제품 컷을 골라 `pnv.webp` 식으로 이름 붙인다.

## ⚠️ 주의 (엑셀에도 표시됨)
- **REGEN PEEL**: "의료 전문가 전용" 문구 필수.
- **GANA KA**: Kojic 4% — EU 비적합, 非EU 시장만.
- **GANA GYNO**: "Vaginal whitening" 표현 금지 → "Intimate brightening & pH care".
- **인증**: FDA 2종(TM·TOX), CPNP 7종만 확정. GMP·ISO는 가격표 근거 없음 → 본사 확인.
- **확인필요 3종**(DM·V LINE·BODY): 본사 판매여부 결정 후 확정.
